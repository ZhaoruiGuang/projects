# vue_share
一个基于vue的自定义分享组件

目前只支持分享到新浪微博和qq空间，后续还会增加微信扫码分享功能，敬请期待！


### 参数说明
- shareTitle
    - 字符串或数字类型
    - 默认值：‘外汇通分享’
    - 非必须
- shareUrl
    - 字符串或数字类型
    - 默认值：‘http://www.forex.com.cn’
    - 非必须
